# Polya's Random Walk — 波利亚随机游走

> 交互式演示波利亚遍历定理（1912）：一维/二维随机游走必然返回原点，三维有约 34% 概率永不返回。

**在线演示**：直接打开 [`index.html`](https://github.com/shcter/polya-random-walk/blob/main/index.html) 即可运行。

---

## 功能一览

| 模式 | 说明 |
|-------|------|
| **1D** | 点在水平线上随机往返，Canvas 渲染 |
| **2D** | 点在网格平面上四向游走，Canvas 网格背景 |
| **3D** | Three.js 3D 渲染，鼠标拖拽旋转/滚轮缩放 |

**新增 v2 功能**：
- 📊 **批量游走** — 同时运行 N 条轨迹，统计返回率
- 🎯 **返回率面板** — 3D 模式下实时对比理论值（34%）与实际返回率
- ⏯ **播放控制** — 完成后可暂停/步进/回退单条轨迹
- 📋 **数据导出** — 一键复制轨迹为 CSV 或 JSON

---

## 波利亚遍历定理

```
d = 1, 2  →  返回概率 = 1（常返，必然无限次返回原点）
d ≥ 3     →  返回概率 < 1（瞬过，有非零概率永不返回）
d = 3     →  精确返回概率 ≈ 34.05%
```

输入较大步数（如 2000 步），设置轨迹数 50 条，观察 3D 模式下的返回率是否接近 34%。
